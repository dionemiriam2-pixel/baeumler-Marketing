Baeumler Marketing7

01 — Unternehmer-Mindset

Was mir in der Schule nie jemand beigebracht hat: wie man sich selbstständig macht. Wir lernen, gute Angestellte zu sein — pünktlich kommen, Aufgaben abarbeiten, Noten kriegen. Aber wie man ein eigenes Business aufbaut, was dazugehört, wie man sich organisiert? Das bringt dir keiner bei. Und genau da stand ich auch. Und wahrscheinlich stehst du gerade an einem ähnlichen Punkt.

Ich sage dir ganz ehrlich: Selbstständigkeit ist nicht für jeden. Nicht weil es zu schwer wäre, sondern weil es Disziplin braucht. Du musst dir Zeit nehmen — regelmäßig, nicht nur wenn du Lust hast. Du musst abends nach der Arbeit nochmal den Laptop aufmachen, wenn du eigentlich lieber auf der Couch liegen würdest. Du musst Dinge lernen, die dir erstmal komplett fremd sind. Und du musst akzeptieren, dass es nicht beim ersten Mal perfekt sein wird.

Aber wenn du bereit bist, das zu tun, dann hast du einen riesigen Vorteil gegenüber allen, die es nicht tun.

Selbstständigkeit ist wie in ETFs investieren

Am Anfang brauchst du ein ganzes Setup und Grundwissen. Dann musst du regelmäßig reinvestieren — Zeit, Energie, manchmal auch Geld. Und irgendwann wächst es exponentiell. Aber nur, wenn du den Anfang machst und dranbleibst. Genau wie bei einer Geldanlage: Die meisten scheitern nicht, weil sie zu wenig wissen, sondern weil sie zu früh aufhören.

Wie viel Zeit brauchst du wirklich?

Du brauchst nicht 8 Stunden am Tag. Aber du brauchst Regelmäßigkeit. Ein, zwei, drei Stunden abends können reichen, wenn du sie konsequent nutzt. Hier ist ein realistischer Plan:

Woche 1

Workspace aufsetzen (dieses Workbook)

Ca. 15–20 Stunden verteilt auf 5–7 Tage. Das klingt viel, aber du arbeitest dich hier durch alles durch: Marke, Website, E-Mail, CRM, Shop. Danach steht dein Fundament.

Woche 2–3

Website verfeinern, Content erstellen

Ca. 5–10 Stunden pro Woche. Texte anpassen, eigene Bilder einfügen, vielleicht den ersten Blog-Artikel schreiben. Alles mit Claude als Unterstützung.

Woche 4

Social Media starten, erstes Produkt live

Ca. 5–10 Stunden. Instagram und Facebook einrichten, erste Posts erstellen, dein erstes digitales Produkt auf Gumroad hochladen.

Ab Monat 2

Regelmäßig dranbleiben

Ca. 3–5 Stunden pro Woche reichen. Content erstellen, Community aufbauen, optimieren. Ab jetzt wird es Routine — und es macht sogar Spaß, weil du siehst, dass es funktioniert.

Baeumler Marketing13

03 — Claude AI verstehen

Ich habe in meinem Leben noch nie eine Zeile Code geschrieben. Ich wusste nicht, was Frontend und Backend bedeutet, was HTML ist, was CSS macht — diese ganzen Begriffe waren mir komplett fremd. Ich konnte eine Website benutzen, aber ich hatte null Ahnung, wie man eine baut.

Dann habe ich mir auf YouTube und TikTok ein paar Videos angeschaut — einfach „Claude Code Website bauen" eingegeben — und gesehen, wie jemand einer KI in normalem Deutsch sagt, was er haben will, und die KI baut eine komplette Website. Innerhalb von Minuten. Ich konnte es nicht glauben.

Claude AI hat drei Modi. Jeder davon hat eine andere Aufgabe, und du wirst im Laufe dieses Workbooks alle drei nutzen. Lass mich dir erklären, was jeder Modus macht und wofür du ihn brauchst:

Modus 1

Claude Chat

Das ist der Modus, den du wahrscheinlich als Erstes nutzen wirst. Hier stellst du Fragen, lässt Texte schreiben, arbeitest an Strategien. Du kannst Projekte anlegen und dort Dateien hochladen — zum Beispiel dein Logo, deine Farbpalette, eine Beschreibung deines Business. So kennt Claude bei jedem neuen Chat deinen Kontext und du musst nicht jedes Mal alles von vorne erklären. Nutze Chat für: Texte schreiben, Ideen brainstormen, Fragen stellen, Prompts vorbereiten, Recherche.

Modus 2

Claude Cowork

Das hat mich am meisten beeindruckt. Cowork ist wie ein Assistent, der an deinem Computer sitzt und Aufgaben für dich erledigt. Du sagst Claude, was er tun soll, und er macht es — im Browser, vor deinen Augen. Wenn du den Browser-Zugriff akzeptierst, kann Claude Websites öffnen, Formulare ausfüllen, Einstellungen ändern. Mit Konnektoren verbindest du Tools wie Canva, Google oder HubSpot direkt mit Claude. Nutze Cowork für: HubSpot einrichten, E-Mail-Signaturen erstellen, Canva-Designs anlegen, repetitive Aufgaben automatisieren.

Modus 3

Claude Code

Dein persönlicher Programmierer. Claude Code arbeitet direkt in deinen Dateien in VS Code — dem Programm, das Entwickler normalerweise zum Coden benutzen. Du sagst in ganz normalem Deutsch, was du haben willst, und Claude schreibt den Code dafür. Website, Cookie-Banner, E-Mail-Signatur, interaktive Elemente — alles. Claude sieht dein ganzes Projekt und baut konsistent weiter. Das Wichtigste: Du musst keinen Code verstehen oder lesen können. Du redest, Claude baut.

Das Wichtigste, was ich gelernt habe

Alles, was du nicht weißt — jede einzelne Frage, die du hast — kannst du einfach in Claude eingeben. „Was ist ein DNS-Record?" „Wie richte ich eine Domain ein?" „Erkläre mir, was Frontend bedeutet." Claude erklärt dir alles in einfacher Sprache.

Schritt 4

Ordnerstruktur verstehen

Claude wird automatisch die richtigen Dateien erstellen. Aber damit du verstehst, was passiert: index.html = deine Startseite. style.css = das Design (Farben, Schriften). script.js = interaktive Funktionen. images/ = ein Ordner für deine Bilder.

Keine Angst vor Code

Ich sage es nochmal, weil es so wichtig ist: Du musst keinen Code lesen oder verstehen können. Du sagst Claude einfach in ganz normalen Sätzen, was du willst. „Mach die Überschrift größer." „Ändere die Hintergrundfarbe zu Weiß." „Füge ein Kontaktformular ein." Claude versteht dich und ändert den Code für dich. Du siehst nur das Ergebnis im Browser.

Baeumler Marketing18

05 — Website-Template

Ich wollte eine richtig gute Website. Nicht so eine Standard-Baukasten-Seite wie bei Wix oder Jimdo, wo man sofort sieht, dass es ein Baukasten ist. Sondern etwas Modernes, Interaktives, Professionelles. Das Problem: Wenn du das von einer Agentur bauen lässt, zahlst du schnell 5.000 Euro aufwärts. Die Lösung: Du nimmst ein professionelles Template als Grundlage und lässt Claude den Rest machen.

Schritt 1

Template-Seiten durchstöbern

Gehe auf Seiten wie 21st.dev, html5up.net oder templatemo.com und such dir ein Template aus, das dir gefällt. Achte auf: modernes Design, responsive (sieht auf Handy gut aus), sauberer Aufbau.

Schritt 2

HTML-Quellcode herunterladen

Lade den HTML-Code des Templates herunter. Bei den meisten Seiten ist das ein ZIP-Ordner mit HTML, CSS und Bild-Dateien.

Schritt 3

In deinen VS Code Ordner einfügen

Entpacke die Dateien und kopiere sie in deinen Projektordner (den du in Kapitel 4 erstellt hast). Öffne den Ordner in VS Code.

Schritt 4

Claude anpassen lassen

Starte Claude Code im Terminal und sag ihm, was du willst:

Prompt: Website anpassen
Ich habe dieses HTML-Template in meinem Ordner. Ich möchte eine ähnliche Website für mein Business „[Name]". Meine Farben sind [Farben], mein Stil ist [modern/minimalistisch/...]. Erstelle mir daraus eine komplette Website mit: Startseite mit Hero-Sektion, Über-mich-Seite, Leistungen/Produkte, Blog, Kontaktformular. Alles auf Deutsch, responsive für mobile Geräte.
Trying and Error — das gehört dazu

Deine Website wird beim ersten Versuch nicht perfekt aussehen. Das ist völlig normal — meins sah am Anfang auch nicht so aus wie jetzt. Der Trick ist: Du sagst Claude einfach, was dir nicht gefällt. „Mach den Hero-Bereich größer." „Die Schrift gefällt mir nicht, nimm eine modernere." „Das Kontaktformular soll auf einer eigenen Seite sein." Claude passt alles an — so oft du willst, ohne dass es extra kostet.

Ich habe am Anfang viel zu viel Zeit mit dem Markennamen verbracht. Stundenlang Schreibweisen verglichen, geschaut ob es den schon gibt, Markenrecht recherchiert, alternative Namen ausprobiert. Heute sage ich: Einfach soll einfach sein. Ich heiße Bäumler, ich mache Marketing — also Bäumler Marketing. Das war's. Kein ausgefallener Fantasiename, kein Branding-Workshop für 3.000 Euro.

Mein Tipp: Nimm deinen Namen und das, was du machst. Prüf, ob die Domain frei ist. Wenn ja — fertig. Du kannst den Namen später immer noch ändern oder erweitern. Aber blockiere dich nicht am Anfang damit.

Schritt 1

Markennamen festlegen

Kurz, einprägsam, ehrlich. Tipp: Dein Name + was du machst. Prüfe, ob die .de-Domain noch frei ist (bei ionos.de).

Schritt 2

Logo in Canva erstellen

Gehe auf canva.com (kostenlos reicht, Pro spart Zeit). Such nach „Logo" und passe eine Vorlage an — oder erstelle eins von Null. Du brauchst dein Logo als SVG (für die Website) und als PNG (für Social Media, E-Mail etc.).

Schritt 3

Brand-Farben & Schriften festlegen

Wähle 2–3 Farben und 2 Schriftarten. Das reicht für den Anfang. Tipp: Schau dir deine Lieblings-Websites an — welche Farben gefallen dir?

Dein Branding:

Markenname:
Hauptfarbe: ________ · Akzentfarbe: ________ · Textfarbe: ________
Schrift Überschriften: ________________ · Schrift Fließtext: ________________
Canva mit Claude verbinden

In Claude Cowork kannst du Canva als Konnektor verbinden. Dann kann Claude direkt in Canva Designs erstellen, Logos anpassen und Vorlagen generieren — ohne dass du Canva selbst öffnen musst.

Kostenlose Bildquellen für deine Website: Unsplash, Pexels, Pixabay — hochwertige Fotos, komplett kostenlos.

Baeumler Marketing26

09 — GitHub + Vercel

Schritt 1

GitHub-Konto erstellen

Gehe auf github.com und erstelle ein kostenloses Konto. GitHub ist wie eine Cloud für deinen Code — jede Änderung wird gespeichert und du kannst jederzeit zurückgehen.

Schritt 2

Repository anlegen & Code hochladen

Erstelle ein neues Repository (= Projektordner auf GitHub). Claude Code kann den Upload für dich übernehmen — sag einfach: „Lade meinen Code auf GitHub hoch."

Schritt 3

Vercel-Konto erstellen & verbinden

Gehe auf vercel.com und melde dich mit deinem GitHub-Konto an. Klicke auf „Import Project" und wähle dein Repository. Vercel erkennt automatisch dein Projekt.

Schritt 4

Domain in Vercel hinzufügen

In Vercel: Settings → Domains → deine IONOS-Domain hinzufügen. Vercel zeigt dir, welche DNS-Einstellungen du bei IONOS ändern musst. SSL (https://) wird automatisch aktiviert — kostenlos.

Das Magische an diesem Setup

Das ist der Moment, wo es richtig cool wird: Ab jetzt gilt — jede Änderung, die du in VS Code machst und auf GitHub hochlädst, ist automatisch in Sekunden auf deiner Website live. Keine manuelle Veröffentlichung, kein FTP-Upload, kein Server-Stress. Du speicherst, lädst hoch, und deine Website aktualisiert sich von alleine. So arbeiten auch professionelle Entwickler — und dank Claude kannst du das jetzt auch, ohne Entwickler zu sein.

10 — E-Mail einrichten

Schritt 1

Google Workspace abonnieren

Gehe auf workspace.google.com → Business Starter Plan (~6€/Monat). Du bekommst: Gmail mit eigener Domain, Google Drive (30GB), Google Kalender, Google Meet.

Schritt 2

Domain verifizieren

Google führt dich durch den Prozess. Du musst einen TXT-Record bei IONOS hinzufügen. Google zeigt dir den genauen Wert — du kopierst ihn einfach bei IONOS in die DNS-Einstellungen.

DNS Records bei IONOS ändern

Bei IONOS gehst du zu: Domains → deine Domain → DNS-Einstellungen. Dort trägst du die MX-Records ein, die Google dir gibt. MX-Records sagen dem Internet: „E-Mails für diese Domain gehen an Google."

Baeumler Marketing 6

01 — Claude einrichten

Was ist Claude?

Claude ist ein KI-Assistent von Anthropic. Anders als andere KI-Tools kann Claude nicht nur Texte schreiben – er programmiert, strukturiert und denkt mit. Claude Code arbeitet direkt in deinen Dateien, sieht den Kontext deines ganzen Projekts und baut konsistent weiter.

Schritt 1

Claude-Konto erstellen

Gehe auf claude.ai und erstelle ein kostenloses Konto. Du brauchst nur eine E-Mail-Adresse.

Screenshot: claude.ai Registrierung
Schritt 2

Claude Pro aktivieren

Upgrade auf Claude Pro (~20$/Monat). Damit bekommst du deutlich mehr Nachrichten pro Tag und Zugang zu den leistungsstärksten Modellen – das ist essenziell für die Website-Erstellung.

Schritt 3

Claude Code installieren

Claude Code ist die Erweiterung, die direkt in deinem Code-Editor arbeitet. So installierst du es: Öffne das Terminal und gib ein: npm install -g @anthropic-ai/claude-code

Warum Claude und nicht ChatGPT?

Claude denkt mit. Wenn du sagst „mach die Seite SEO-freundlich", fügt Claude nicht nur Meta-Tags ein – es erstellt Schema-Markup, eine Sitemap und erklärt dir warum. Claude Code sieht dein ganzes Projekt und baut konsistent weiter.

02 — VS Code & Projekt

Schritt 1

VS Code herunterladen

Gehe auf code.visualstudio.com und lade die kostenlose Version für dein Betriebssystem herunter. Die Installation dauert nur wenige Minuten.

Screenshot: VS Code Download-Seite

Erstelle auf deinem Computer einen neuen Ordner für dein Website-Projekt. Zum Beispiel: Desktop → mein-business-website. Öffne diesen Ordner in VS Code über Datei → Ordner öffnen.

Keine Angst vor Code

Du musst keinen Code verstehen oder schreiben können. Claude macht das alles für dich. Du sagst nur, was du willst – in ganz normaler Sprache. Zum Beispiel: „Erstelle mir eine Startseite mit einem großen Bild oben, meinen Leistungen und einem Kontaktformular."

03 — Website bauen lassen

Schritt 1

Startseite beschreiben

Sag Claude in normaler Sprache, was du brauchst. Beispiel: „Erstelle mir eine Startseite mit Hero-Sektion, meinen Leistungen, Testimonials und einem Kontaktformular. Farben: Weiß, Dunkelgrau, Coral als Akzent."

Design-System festlegen

Schritt 3

Unterseiten erstellen

SEO-Optimierung

Iterieren statt perfektionieren

Deine Website muss nicht beim ersten Mal perfekt sein. Starte mit einer soliden Basis und verbessere sie Schritt für Schritt. Jede Änderung kannst du Claude einfach sagen: „Mach den Button größer", „Ändere die Überschrift", „Füge einen Blog-Bereich hinzu".

Baeumler Marketing 12

04 — GitHub einrichten

Schritt 1

GitHub-Konto erstellen

Gehe auf github.com und erstelle ein kostenloses Konto. GitHub ist der Standard für Code-Verwaltung – Millionen von Entwicklern nutzen es täglich.

Schritt 2

Repository anlegen

Ein Repository (kurz: Repo) ist dein Projektordner auf GitHub. Erstelle ein neues Repo mit dem Namen deiner Website (z.B. „meine-website").

Screenshot: Neues Repository auf GitHub
Schritt 3

Code hochladen

Verbinde dein lokales Projekt mit GitHub. Claude kann dir die Befehle dafür geben – oder du nutzt die GitHub-Integration in VS Code. Ein Klick und dein Code ist gesichert.

Warum GitHub wichtig ist

Wenn du etwas kaputt machst, kannst du jederzeit zu einer früheren Version zurückkehren. Und: Vercel (dein Hosting) liest direkt von GitHub – jede Änderung, die du hochlädst, ist automatisch Sekunden später live auf deiner Website.

Baeumler Marketing 13
05

Mit Vercel
live schalten

Vercel ist dein kostenloses Hosting. Du verbindest es einmal mit GitHub – und ab dann ist jede Änderung in Sekunden online. Kein FTP, kein Server-Stress.

Baeumler Marketing 14

05 — Vercel Hosting

Schritt 1

Vercel-Konto erstellen

Gehe auf vercel.com und melde dich mit deinem GitHub-Konto an. Die Verbindung zwischen beiden Diensten wird automatisch hergestellt.

Schritt 2

Projekt importieren

Klicke auf „Import Project" und wähle dein GitHub-Repository aus. Vercel erkennt automatisch, was für ein Projekt es ist, und konfiguriert alles für dich.

Screenshot: Vercel Projekt-Import
Schritt 3

Deployment starten

Klicke auf „Deploy" – und innerhalb von Sekunden ist deine Website live. Du bekommst eine URL wie dein-projekt.vercel.app. Ab jetzt wird jeder Push auf GitHub automatisch veröffentlicht.

Kostenlos und schnell

Vercel bietet ein großzügiges kostenloses Kontingent. Für die meisten Business-Websites reicht das völlig aus. Deine Seite wird weltweit über ein CDN ausgeliefert – superschnell, egal wo deine Besucher sitzen.

06 — Domain verbinden

Schritt 1

Domain registrieren

Gehe auf ionos.de und registriere deine Wunschdomain. Eine .de-Domain kostet ca. 1€/Monat. Tipp: Halte den Namen kurz, einprägsam und passend zu deinem Business.

Schritt 2

DNS-Einstellungen anpassen

In den IONOS-Einstellungen musst du die Nameserver auf Vercel umstellen. Vercel zeigt dir genau welche Werte du eintragen musst.

Schritt 3

Domain in Vercel hinzufügen

In deinem Vercel-Dashboard gehst du zu Settings → Domains und fügst deine neue Domain hinzu. Das SSL-Zertifikat (https://) wird automatisch erstellt – kostenlos.
